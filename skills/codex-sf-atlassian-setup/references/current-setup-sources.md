# Current Setup Sources

Use these URLs as the source of truth for this skill's setup flow.

- Salesforce CLI download page: https://developer.salesforce.com/tools/salesforcecli
- Salesforce `sf org login web` command reference: https://github.com/salesforcecli/plugin-auth
- Atlassian MCP getting started: https://support.atlassian.com/atlassian-rovo-mcp-server/docs/getting-started-with-the-atlassian-remote-mcp-server/
- Atlassian OAuth 2.1 setup: https://support.atlassian.com/atlassian-rovo-mcp-server/docs/configuring-oauth-2-1/

## Volatile Note

- Use `https://mcp.atlassian.com/v1/mcp`.
- Do not use `/v1/sse` for new setup. Atlassian documents that `/v1/sse` is deprecated after June 30, 2026.
